### Functions
---

A function is a block of code that does a specific job, and you can use it repeatedly without rewriting it. 

def greet():
    print("hello!")
greet() # prints hello
greet() #prints "hello" again

def - I am defining a function
greet - is the name you give it
() - are where your inputs go (empty for now)
The indented code underneath is what the function does. 
Calling greet() runs that code


#### Categores

##### 1. Built-in functions
These come with python. 

For example:
print("Hello)
len("hello)
input("Name: ")
range(5)

You never defined these. Python just has them ready.

##### 2. User-defined functions 
These are functions we create like the one above
**Static Functions**
Always does the same thing. No inputs, no changes. Same result every time

** Dynamic Function**
Changes its behavior based on what you give it.
Takes paramters, behaviour changes based on input

Parameters are the inputs you put inside the parentheses when you define a function. Parameters are the blanks (a.b), arguments are the actual values you fill n.


##### 3. Module/library functions
These live inside modules or packages. You import them first, then use them
