### Functions
---

A function is a block of code that does a specific job, and you can use it repeatedly without rewriting it. 

def greet():
    print("hello!")
greet() # prints hello
greet() #prints "hello" again

def - I am defining a function
greet - is the name you give it
() - are where your inputs go (empty for now)
The indented code underneath is what the function does. 
Calling greet() runs that code
